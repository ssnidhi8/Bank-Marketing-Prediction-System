# Bank Marketing Prediction System

## Project Overview
The Bank Marketing Prediction System is a machine learning application that predicts whether a customer will subscribe to a bank term deposit. The system analyzes customer information such as age, job, balance, marital status, and previous campaign outcomes to make predictions.

## Features
- Customer subscription prediction using Machine Learning
- Interactive Streamlit web application
- Data visualization and analysis
- User-friendly interface

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Streamlit
- Plotly

## Dataset
The project uses the Bank Marketing dataset which contains customer demographic and campaign interaction information.
